📖 Overview

This project simulates a Traffic Light Control System using Arduino Uno in Proteus.
The system automatically controls Red, Yellow, and Green LEDs according to predefined timing intervals to mimic a real traffic signal. An additional LED connected to pin D5 can be manually controlled using an ON/OFF switch.
The project demonstrates Arduino digital output control, timing operations, LED interfacing, and basic traffic signal automation.

⚙️ Working Principle

1. Green LED turns ON for a specified duration.
2. Yellow LED turns ON to indicate transition.
3. Red LED turns ON to stop traffic.
4. The cycle repeats continuously.
5. The additional LED connected to D5 can be manually controlled using an ON/OFF switch.
